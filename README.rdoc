== Client Portal RoR app

>this is a project that was built for a client who needed a postal for their customers

This app allows a user to do the following
	* Create Users
	    * internal users -> have access to all files
	    * customer users -> have access to files with matching customer id
		
	* Create Customers
	
	* Create Location - associated with customers
	
	* Upload/Download Files  - associated with Customers, Users and Locations
